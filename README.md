# mpdweb.frontend

This repository houses the frontend part of [MPDWeb](https://github.com/toncherami/mpdweb) â€“ a web client for the [Music Player Daemon](https://musicpd.org).

## Stack

- React
- TypeScript
- Ramda
- SASS
- Babel
- Webpack

## Development

Make sure you have the [backend](https://github.com/toncherami/mpdweb.backend) listening on port `8989`.

To get the local development server up and running first install the dependencies:

```shell
yarn install
```

Then start it:

```shell
yarn run start
```

And open http://localhost:8080 to see the app.
